SatScanner for DreamOS 2.6

Satellite signal finder and transponder scanner.

Install (one line)

wget -O - https://xfayez95.github.io/satscanner/installer.sh | sh

Manual install

Download the latest package and install it with dpkg:

https://github.com/xFayez95/satscanner/releases/latest/download/satscanner-dreamos.deb
dpkg -i satscanner-dreamos.deb
